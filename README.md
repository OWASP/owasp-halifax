# Building The Site

__Download:__
```bash
git clone https://github.com/owasp/owasp-halifax.git
cd owasp-halifax/
```

__Build and Serve Locally:__
```bash
make
make serve
```

__Docker Build:__
```bash
sudo docker build -t owasp-halifax .
```

__Docker Compose:__
```bash
sudo docker-compose up .
```
